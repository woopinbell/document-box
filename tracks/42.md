# 42 학습 경로

Linux/Git 선수 과정을 마치면 C·C++·인프라 세 갈래가 열린다. 한 갈래씩 진행해도 되고 서로 다른
작업 공간에서 함께 진행해도 된다. 세 갈래를 모두 끝내야 Web 갈래로 합류할 수 있다.

```text
Linux/Git
├─ C1 → C2 → C3 → C4 → C5 → C6 → C7
├─ CPP1 → CPP2 → CPP3
└─ INFRA1
        ↓ 모두 완료
JOIN1 → JOIN2 → FINAL
```

각 카드의 공통 진행법과 연습문제 선택법은 [학습 시작](README.md)에서 한 번만 읽는다. Git의 시작
커밋이나 별도 작업 공간이 헷갈리면 [기술 안내](TECHNICAL_GUIDE.md)를 참고한다.

시작 전에 Document Box에서 다음을 실행한다.

```sh
make preflight TRACK=42
```
